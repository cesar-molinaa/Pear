# Pear# pear-website
I'm actually building a website inspired in Apple, using a pear instead an apple for the logo. It's one of my first websites, so I'm doing it not only for fun, also for learning (and it's working because I have learned a lot of things). The web consists of different sections: the presentation, news (with the latest products), the different products of the company, a carrousel with different IPears, some innovation actions the company is doing, a description of the headquarters of Pear, a brief description of the company and its objectives, the "green" goals of the company to help the environment, some opinions of users and finally a footer with the main information. I tried to implement all my knowledge about HTML, CSS and JS. I've hardly used JS because right now it's difficult to me to write it, but I understand it. I have learned some new stuff with CSS too, doing the web prettier (I think).
